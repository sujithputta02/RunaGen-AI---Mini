CERTIFICATION CATEGORIZATION - PROPER DISTINCTION
================================================================================

PROBLEM:
--------
"Microsoft Imagine Cup 2026" was being detected as a certification, but it's 
actually a competition/hackathon, not a professional certification.

The system was treating everything as "certifications" without distinguishing:
- Professional Certifications (AWS, Azure, PMP)
- Competitions/Hackathons (Imagine Cup, Code Jam)
- Online Courses (Coursera, Udemy)

SOLUTION:
---------
Enhanced certification analysis to properly categorize into 3 types:

1. PROFESSIONAL CERTIFICATIONS (Real Value)
   ✓ Cloud: AWS Certified, Azure Certified, GCP Certified
   ✓ Project Management: PMP, PRINCE2, CAPM, CSM, PSM
   ✓ Agile: Scrum Master, Product Owner, SAFe
   ✓ Security: CISSP, CEH, CompTIA Security+, CISM, CISA
   ✓ Networking: CCNA, CCNP, CompTIA Network+
   ✓ Database: Oracle Certified, MongoDB Certified
   ✓ Programming: Oracle Java, Microsoft Certified, Red Hat
   ✓ Data: Databricks, Snowflake, Tableau Certified

2. COMPETITIONS & HACKATHONS (Good but Different)
   ✓ Hackathon
   ✓ Microsoft Imagine Cup
   ✓ Google Code Jam
   ✓ Kaggle Competition
   ✓ Coding Competition
   ✓ Programming Contest
   ✓ Tech Challenge
   ✓ Innovation Challenge
   ✓ Startup Competition

3. ONLINE COURSES (Learning but Not Certifications)
   ✓ Coursera
   ✓ Udemy
   ✓ edX
   ✓ Udacity
   ✓ Pluralsight
   ✓ LinkedIn Learning
   ✓ DataCamp
   ✓ Codecademy

WHAT'S TRACKED NOW:
------------------

analysis['professional_certifications'] = {
    'count': 2,
    'found': ['aws certified', 'pmp']
}

analysis['competitions_hackathons'] = {
    'count': 1,
    'found': ['imagine cup']
}

analysis['online_courses'] = {
    'count': 3,
    'found': ['coursera', 'udemy', 'datacamp']
}

analysis['has_certifications'] = True  # Based on professional certs only
analysis['has_competitions'] = True
analysis['has_courses'] = True
analysis['total_achievements'] = 6  # Sum of all three
