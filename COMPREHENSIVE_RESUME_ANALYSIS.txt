-------------------------------------------

1. SECTION ANALYSIS ✓

   - Summary/Objective
   - Work Experience
   - Education
   - Skills
   - Projects
   - Certifications
   - Achievements/Awards
   
   Scores: Section count out of 8

2. WORK EXPERIENCE ANALYSIS ✓
   - Counts number of job positions

   - Verifies work history exists
   - Checks for career progression
   
   Patterns detected:
   - "2020-2023"
   - "2020-Present"
   - "Jan 2020 - Dec 2023"

3. ACTION VERBS ANALYSIS ✓
   Checks for 37 strong action verbs:
   - achieved, administered, analyzed, architected, automated
   - built, collaborated, created, delivered, designed
   - developed, directed, drove, enhanced, established
   - executed, generated, implemented, improved, increased
   - initiated, launched, led, managed, optimized
   - orchestrated, organized, pioneered, planned, produced
   - reduced, redesigned, resolved, spearheaded, streamlined
   - transformed, upgraded
   
   Counts: How many different action verbs used

4. QUANTIFIABLE ACHIEVEMENTS ✓
   Detects ALL types of metrics:
   - Percentages: "50%", "increased by 40%"
   - Dollar amounts: "$50K", "$1M", "$500,000"
   - Time periods: "5 years", "3+ months"
   - Scale: "1000+ users", "500 customers"
   - Multipliers: "2x", "10x improvement"
   - Large numbers: "1 million", "500 thousand"
   - Impact statements: "increased by 50", "reduced by 30%"
   
   Counts: Total quantifiable metrics found

5. KEYWORD DENSITY ANALYSIS ✓
   - Checks how often required skills appear
   - Tracks which keywords are mentioned
   - Calculates average keyword frequency
   - Identifies missing keywords
   
   Details: Keyword-by-keyword breakdown

6. EDUCATION ANALYSIS ✓
   Checks for education indicators:
   - Degrees: Bachelor, Master, PhD, MBA
   - Abbreviations: B.S, M.S, B.Tech, M.Tech
   - Institutions: University, College, Institute
   
   Verifies: Education section exists and is complete

7. PROJECTS ANALYSIS ✓
   Looks for project indicators:
   - "project", "developed", "built", "created"
   - "github", "portfolio", "demo", "repository"
   - Technical implementations
   
   Counts: Number of project mentions

8. CERTIFICATIONS ANALYSIS ✓
   Detects certification keywords:
   - "certified", "certification", "certificate"
   - Cloud: AWS, Azure, GCP
   - Methodologies: PMP, Scrum, Agile
   
   Counts: Certification mentions

9. RESUME LENGTH ANALYSIS ✓
   Measures:
   - Word count (ideal: 300-1500 words)
   - Character count
   - Line count
   - Overall length appropriateness
   
   Validates: Resume is not too short or too long

10. CONTACT INFORMATION ✓
    Verifies presence of:
    - Email address (pattern matching)
    - Phone number (various formats)
    - LinkedIn profile
    - GitHub profile
    
    Ensures: Contact info is complete

11. COMPREHENSIVE FORMATTING SCORE ✓
    Weighted scoring (100 points total):
    - Section completeness: 30 points
    - Action verbs: 15 points
    - Quantifiable results: 25 points
    - Keyword density: 15 points
    - Resume length: 10 points
    - Contact info: 5 points
