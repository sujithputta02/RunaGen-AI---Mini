# Quick Start: PDF Viewer Feature

## What It Does
Shows your uploaded PDF resume on the webpage alongside the AI analysis results.

## How to Use

### 1. Upload Resume
```
Click upload zone → Select PDF → File appears
```

### 2. Analyze
```
Click "Start Analysis" → Wait for processing
```

### 3. View Results
```
Scroll to results → PDF viewer at top → Analysis below
```

### 4. Navigate PDF
```
← Previous | Next → : Switch pages
- | + : Zoom out/in
```

## Controls

| Button | Action |
|--------|--------|
| ← Previous | Go to previous page |
| Next → | Go to next page |
| - | Zoom out (min 50%) |
| + | Zoom in (max 300%) |

## Troubleshooting

### PDF Not Showing
- ✅ Check file is PDF format
- ✅ Refresh page
- ✅ Check browser console

### Slow Rendering
- ✅ Reduce zoom level
- ✅ Use smaller PDF
- ✅ Close other tabs

### Controls Not Working
- ✅ Wait for page to load
- ✅ Check JavaScript enabled
- ✅ Try different browser
